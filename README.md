# 📱 RedInfancia

Aplicación desarrollada en Flutter para la gestión de niños en hogares comunitarios.

---

## 🚀 Tecnologías utilizadas

* Flutter
* Supabase (base de datos y almacenamiento)
* Dart

---

## ⚙️ Configuración del proyecto

### 1. Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/redinfancia.git
cd redinfancia
```

---

### 2. Instalar dependencias

```bash
flutter pub get
```

---

### 3. Configurar Supabase

Ir al archivo:

```
lib/main.dart
```

y colocar:

```dart
const supabaseUrl = 'TU_URL';
const supabaseKey = 'TU_KEY';
```

---

### 4. Ejecutar la aplicación

```bash
flutter run
```

---

## 📂 Funcionalidades actuales

* Login de usuarios
* Registro de niños
* Subida de imágenes
* Conexión con Supabase
